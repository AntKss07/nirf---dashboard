# NIRF Dashboard

This project is a web-based dashboard for visualizing the National Institutional Ranking Framework (NIRF) 2025 data. It allows users to explore the top-ranked institutions in India across various categories and regions.

## Features

*   **Interactive Filtering:** Filter institutions by category and region.
*   **Top 10 Institutes:** View the top 10 institutions in a selected category.
*   **Regional Distribution:** See the distribution of institutions across different regions in a pie chart.
*   **Score Distribution:** Analyze the score distribution by category using a box plot.
*   **"All" Category:** View combined data for all categories.

## How to Run

1.  **Install the dependencies:**
    ```bash
    pip install dash pandas dash-bootstrap-components waitress
    ```

2.  **Run the application:**
    ```bash
    python dashboard.py
    ```

3.  **Open your web browser** and navigate to `http://localhost:8050`.
